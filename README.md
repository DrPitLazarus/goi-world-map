# goi-world-map

An online version of GOI's world map powered by Konva.js. Currently a WIP.

## Dev
Install parcel-bundler
```bash
$ yarn global add parcel-bundler
# or
$ npm install -g parcel-bundler
```

Install dependencies
```bash
$ yarn
# or
$ npm install
```

Start
```bash
$ yarn run dev
# or
$ npm run dev
```
